# bai_thuc_hanh_4_yc_5
 Đọc dataset CSV → sinh báo cáo PDF

1. download install https://wkhtmltopdf.org/downloads.html
2. Thêm C:\Program Files\wkhtmltopdf\bin vào path của system variable
3. install pdfkit: pip install pdfkit
4. Python packages: pip install flask pandas matplotlib pdfkit
5. Run app: py app.py
6. Link truy cập ứng dụng: http://127.0.0.1:5000/
